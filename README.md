### Запуск проектов

1. **Загрузка проекта**:
   - Выполните команду:
     ```
	git clone https://github.com/Minaeff1q/IoT.git
     ```

2. **Перейдите в директорию проекта****:
   - Выполните команду:
     ```
    cd IoT
     ```
	 
3. **Запуск проекта**:
   - Выполните команду:
     ```
     docker-compose up --build
     ```
